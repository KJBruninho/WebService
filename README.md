# 🌐 WebService - Inovacorp

Este repositório contém o **WebService** desenvolvido para a Inovacorp. Trata-se de uma solução robusta em Java projetada para fornecer serviços web eficientes, integrando lógica de negócio com interfaces acessíveis.

## Funcionalidades

  * **API RESTful:** Endpoints estruturados para comunicação de dados.
  * **Integração Java:** Core da aplicação desenvolvido em Java para alta performance e escalabilidade.
  * **Interface Web:** Páginas HTML integradas para monitorização ou interação básica com o serviço.
  * **Gestão de Dados:** Processamento e disponibilização de informações corporativas.

## 🛠️ Tecnologias Utilizadas

  * **Linguagem Principal:** Java 
  * **Frontend/Doc:** HTML 
  * **Ferramenta de Build:** Maven

## Instalação e Configuração

1.  **Clonar o repositório:**
    git clone [https://github.com/KJBruninho/WebService.git](https://www.google.com/search?q=https://github.com/KJBruninho/WebService.git)

2.  **Aceder ao diretório do projeto:**
    cd WebService/Inovacorp

3.  **Compilar e Executar:**
    mvn clean install
    mvn spring-boot:run

## Estrutura de Pastas

  * WebService/Inovacorp/ - Pasta principal do projeto.
  * src/ - Código fonte Java.
  * resources/ - Configurações e ficheiros estáticos.
  * pom.xml - Gestão de dependências.
